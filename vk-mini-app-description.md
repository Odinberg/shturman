# Штурман — AI-навигатор

**ID приложения**: app54657016

## Краткое описание

Интегративная программа глубинной саморегуляции. 7 направлений самоисследования с AI-ассистентом.

## Полное описание

«Штурман» — это интегративная платформа для системного самоисследования, построенная на семи взаимодополняющих методических направлениях: работа с телом, эмоциональная саморегуляция, когнитивная пересборка, ценностная навигация, поведенческие паттерны, социальное взаимодействие и смысловая интеграция. Каждое направление опирается на доказательные подходы из соматической психологии, когнитивно-поведенческой терапии, теории привязанности и нейрофизиологии стресса.

Приложение разработано для участников закрытого клуба «Архипелаг» — сообщества, ориентированного на осознанное развитие и глубинную трансформацию. Доступ к платформе осуществляется через VK Mini App, что позволяет использовать её непосредственно в социальной сети ВКонтакте без установки дополнительного программного обеспечения.

В основе работы лежит двухконтурная модель: пользователь ежедневно выполняет короткие структурированные практики по выбранному направлению, а AI-ассистент анализирует динамику ответов, выявляет зоны напряжения и предлагает индивидуализированные рекомендации. Система учитывает не только содержание ответов, но и их согласованность, глубину рефлексии и временну́ю динамику. Пользователь получает карту собственных паттернов с отслеживанием прогресса по каждому из семи направлений.

Конфиденциальность является архитектурным принципом: все данные пользователя хранятся в изолированном контуре, обработка AI-ассистентом происходит без передачи данных третьим лицам. Пользователь в любой момент может запросить экспорт или полное удаление своей информации.

## Как работает

1. **Выбор направления** — пользователь определяет приоритетное методическое направление из семи доступных, опираясь на встроенную диагностическую анкету.
2. **Ежедневные практики** — система формирует структурированные задания: дневниковые техники, телесные практики, рефлексивные вопросы и поведенческие эксперименты.
3. **AI-анализ** — ассистент обрабатывает ответы, выявляет повторяющиеся паттерны и формирует аналитическую сводку с рекомендациями.
4. **Бонусы за глубину** — система оценивает качество рефлексии и полноту прохождения практик; при достижении пороговых значений открываются углублённые модули и дополнительные инструменты анализа.

## Ключевые особенности

- **7 методических направлений** — от соматических практик до смысловой интеграции, охватывающих основные уровни саморегуляции.
- **AI-ассистент** — анализирует пользовательские ответы, выявляет паттерны и формирует персонализированные рекомендации без использования шаблонных сценариев.
- **Конфиденциальность** — данные не передаются третьим лицам; полный контроль над информацией остаётся у пользователя.
- **Закрытое сообщество** — платформа объединяет участников клуба «Архипелаг», создавая среду для осознанного развития.

## Технические детали

- **Серверная часть**: FastAPI (Python)
- **Клиентская часть**: Next.js (React)
- **Платформа**: VK Mini App, работает внутри социальной сети ВКонтакте
- **Хранение данных**: изолированное, с шифрованием на уровне базы данных
