# Архимедов рычаг: спецификация интеграции (Режим Хроник)

## Обзор
Режим Хроник — связующий бонус, открывающийся после прохождения ВСЕХ 7 направлений. Это вращающийся 3D-куб, где каждый день за последний месяц представлен как многослойный артефакт со всеми слоями самопознания: эмоция, тело, сценарий, субличность, тень, маленькое событие, ресурсное состояние.

## Принцип работы

### Визуализация: 3D-куб
- **Куб** вращается по свайпу (или автоматически — медленно, медитативно)
- **6 граней** — 6 направлений (7-е — ось времени)
- **На каждой грани** — календарная сетка месяца (30 дней)
- **Каждый день** — точка. При нажатии раскрывается в «слоёный срез»

### Слоёный срез дня
При выборе конкретного дня капитан видит:
```
┌─────────────────────────────────────┐
│  День: 15 марта, вторник            │
├─────────────────────────────────────┤
│ 🧠 Эмоция: 😊 энергия               │
│    (из чекина Направления 2)        │
├─────────────────────────────────────┤
│ 🤸 Тело: 🟢 лёгкость                │
│    (из сенсорного чекина Н5)        │
├─────────────────────────────────────┤
│ 🔁 Сценарий: «Надо» против «хочу»   │
│    (из карты паттернов Н1)          │
├─────────────────────────────────────┤
│ 👤 Субличность: Вдохновленный Я     │
│    (из дневника Н6)                 │
├─────────────────────────────────────┤
│ 🌑 Тень: раздражение на коллегу     │
│    (из голосового разбора Н4)       │
├─────────────────────────────────────┤
│ 🦋 Событие: встретил собаку         │
│    (из копилки чудес Н7)            │
├─────────────────────────────────────┤
│ 💎 Ресурс: «Я справлюсь»            │
│    (из коллекции ресурсов Н2)       │

```

## Техническая реализация

### Фронтенд
- **3D-движок:** Three.js или React Three Fiber
- **Куб:** CSS3D или WebGL
- **Слоёный срез:** модальное окно или разворот грани
- **Анимация:** плавное вращение, параллакс при наведении
- **Данные:** агрегируются через один endpoint

### Бэкенд (агрегация)
```
GET /api/chronicles/day/{date}
→ возвращает все слои за конкретный день

GET /api/chronicles/month/{year}/{month}
→ возвращает матрицу 30×7 (дни × слои) для куба
```

### Данные, которые агрегируются
| Слой | Источник | Тип данных |
|------|----------|------------|
| Эмоция | emotional_checkins | текст + эмодзи |
| Тело | sensory_checkins | enum (5 ощущений) |
| Сценарий | pattern_map вывод | текст (паттерн за период) |
| Субличность | multiplicity_posts | текст (какая субличность писала) |
| Тень | shadow_dialogs | текст (на кого раздражение) |
| Событие | butterfly_events | текст (маленькое событие) |
| Ресурс | resource_states | текст (ресурсная фраза) |

Если за конкретный день нет данных по какому-то слою — слой скрыт (не показывается пустым).

## Бонусная механика
При каждом открытии слоёного среза капитан получает «артефакт дня» — одну случайную фразу из любого слоя, выведенную как красивая карточка для шеринга.

## Приоритет реализации
1. **MVP:** статическая календарная сетка с точками-днями (без 3D)
2. **v1:** 3D-куб с вращением (Three.js)
3. **v1.5:** Слоёный срез дня (модальное окно)
4. **v2:** Артефакты дня (карточки для шеринга)
5. **v2.5:** Экспорт месяца как PDF-книги «Хроники капитана»
